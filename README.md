# Nifty Trading Strategy

Notebook and scripts for a Nifty trading strategy: data collection, signal generation, backtesting, and performance analysis.

## Features
- Collects historical Nifty / Nifty50 data
- Implements trading logic and signal generation
- Backtests strategy and computes performance metrics (CAGR, drawdown, win-rate)
- Saves example trades and result charts

## How to run
1. Clone repo:
   ```bash
   git clone https://github.com/<your-username>/nifty-trading-strategy.git
   cd nifty-trading-strategy
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
